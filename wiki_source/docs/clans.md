# Кланы на сервере SP.A

Кланы — это отличный способ объединиться с друзьями, строить вместе, сражаться и доминировать на сервере!

## Создание клана

Чтобы создать свой клан, вам потребуется определенная сумма игровой валюты (например, 5000 монет). Откройте чат и введите команду:

```
/clan create [название_клана]
```

- `[название_клана]` — это уникальное имя вашего клана.

!!! info "Важно"
    Название клана должно быть уникальным и не нарушать общие правила сервера.

## Управление кланом

После создания клана вы становитесь его лидером и получаете доступ к следующим командам:

### Приглашение игроков

Чтобы пригласить игрока в свой клан:

```
/clan invite [ник_игрока]
```

Приглашенный игрок должен принять приглашение командой `/clan accept [название_клана]`.

### Выгнать игрока

```
/clan kick [ник_игрока]
```

### Повысить/понизить ранг игрока

Вы можете назначать игрокам внутри клана различные ранги (например, офицер), которые дают им дополнительные права.

```
/clan promote [ник_игрока]
/clan demote [ник_игрока]
```

### Передача лидерства

Если вы хотите передать управление кланом другому игроку:

```
/clan transfer [ник_игрока]
```

!!! warning "Внимание!"
    Передача лидерства необратима. Убедитесь, что доверяете игроку.

### Роспуск клана

Если вы хотите распустить клан (только для лидера):

```
/clan disband
```

## Взаимодействие кланов

- **Союзы:** Кланы могут заключать союзы для совместной игры и защиты.
- **Войны:** Кланы могут объявлять войны другим кланам.

Подробности о командах для союзов и войн можно узнать в справочнике команд.
