# Приват территорий

На нашем сервере используется плагин **WorldGuard** для защиты ваших построек. Это надежный способ убедиться, что никто не сможет сломать ваши сундуки и дома.

## Как заприватить территорию?

Процесс привата состоит из двух шагов: выделение региона и его регистрация.

### Шаг 1: Выделение региона

Для выделения вам понадобится **деревянный топор**. Его можно либо скрафтить, либо получить командой `//wand`.

1.  **Первая точка**: Возьмите топор в руку и кликните **левой кнопкой мыши** по блоку в одном углу будущей территории.
2.  **Вторая точка**: Кликните **правой кнопкой мыши** по блоку в противоположном углу по диагонали.

Вы выделили кубоид — параллелепипед, который охватывает вашу территорию от неба до бедрока.

!!! info "Совет"
    Вы можете выделить территорию, даже если блоки находятся в воздухе или под землей.

### Шаг 2: Создание региона

После выделения введите команду для создания региона:

```
/rg claim [название_региона]
```

- `[название_региона]` — это уникальное имя для вашей территории. Используйте только латинские буквы и цифры.

**Пример:** `/rg claim my_sweet_home`

Если все прошло успешно, в чате появится сообщение о создании региона.

## Управление регионом

### Добавление друга в регион

Чтобы ваш друг мог строить на вашей территории, добавьте его в регион:

```
/rg addmember [название_региона] [ник_игрока]
```

### Удаление игрока из региона

```
/rg removemember [название_региона] [ник_игрока]
```

### Просмотр информации о регионе

Чтобы узнать, кто является владельцем региона и кто в него добавлен, используйте команду:

```
/rg info [название_региона]
```

Если вы стоите внутри региона и не знаете его названия, просто введите `/rg info`.
