Как начать играть

1

Купите Minecraft

Для игры на сервере необходима лицензионная версия Minecraft: Java Edition. Приобрести ее можно на официальном сайте.

2

Добавьте сервер

В главном меню игры нажмите "Сетевая игра", затем "Добавить сервер". В поле адреса введите наш IP который вы скопировали выше

* Заранее скачайте необходимый список модов с сервера - клик.

3

Присоединяйтесь!

Найдите наш сервер в списке и нажмите "Подключиться". Добро пожаловать в мир SP.A!
