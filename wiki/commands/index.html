    Справочник команд

    Здесь собран список самых полезных команд, доступных игрокам на сервере. Команды сгруппированы по категориям для удобного поиска.

    Основные команды

    Команда Описание
    /register [пароль] [пароль] Регистрация нового аккаунта на сервере.
    /login [пароль] Вход в свой аккаунт при заходе на сервер.
    /spawn Мгновенная телепортация на спавн.
    /rtp Телепортация в случайную точку основного мира.

    Телепортация

    Команда Описание
    /sethome [название] Установить точку дома в текущей локации.
    /home [название] Телепортироваться в указанную точку дома.
    /delhome [название] Удалить точку дома.
    /call [ник_игрока] Отправить запрос на телепортацию к игроку.
    /tpaccept Принять входящий запрос на телепортацию.
    /tpadeny Отклонить входящий запрос на телепортацию.
    /warp [название] Телепортироваться на публичную точку варпа.
    /warps Посмотреть список доступных варпов.

    Экономика и торговля

    Команда Описание
    /money Посмотреть свой баланс.
    /pay [ник_игрока] [сумма] Перевести деньги другому игроку.
    /shop Открыть графический интерфейс глобального магазина.
    /ah Открыть аукцион.
    /ah sell [цена] Выставить предмет из руки на аукцион.
    /trade [ник_игрока] Отправить запрос на безопасную торговлю.
    /jobs browse Посмотреть список доступных работ.
    /jobs join [название] Устроиться на работу.

    Приват территорий

    Команда Описание
    //wand Получить деревянный топор для выделения региона.
    /rg claim [название] Заприватить выделенную территорию.
    /rg addmember [название] [ник] Добавить игрока в свой регион.
    /rg removemember [название] [ник] Удалить игрока из региона.
    /rg info Посмотреть информацию о регионе, в котором вы стоите.
    /rg list Показать список ваших регионов.

    Кланы

    Команда Описание
    /clan create [название] Создать новый клан.
    /clan invite [ник_игрока] Пригласить игрока в клан.
    /clan accept [название] Принять приглашение в клан.
    /clan kick [ник_игрока] Выгнать игрока из клана.
    /clan disband Распустить свой клан (только для лидера).
    /clan transfer [ник_игрока] Передать лидерство в клане.
    /c [сообщение] Написать в чат своего клана.

    Команды для донат-рангов

    Команда Описание Доступ
    /kit vip Получить набор VIP. VIP и выше
    /feed Мгновенно утолить голод. Premium и выше
    /fly Включить/выключить режим полета. Deluxe и выше
    /workbench Открыть виртуальный верстак. Deluxe и выше
